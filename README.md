Padel Club Admin Hub

This project is an admin management system for a Padel Club, built to manage players, bookings, courts, and club operations efficiently.

📖 About

A centralized admin dashboard for managing Padel club operations including players, courts, and bookings.

🚀 Features
Player management
Court management
Booking system
Schedule tracking
Admin dashboard
Simple CRUD operations
Clean UI structure
